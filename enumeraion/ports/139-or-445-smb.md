# 139 | 445 - SMB

## Mount SMB

```
sudo mount -t cifs -o guest //$TARGET/tmp /mnt/share
sudo mount -t cifs -o guest,vers=1.0 //$TARGET/Test\ Share/ /mnt/share
```

```
df -h
```

```
sudo umount /mnt/share
```

## Enumeration

```
nmap -p 445 --script=smb-enum-shares.nse,smb-enum-users.nse <ip>
```
